# Hugo S. Oliveira — Academic Website

## Deploy

1. Upload to GitHub
2. Go to Settings → Pages
3. Enable GitHub Pages (main branch)

## Update Publications

Replace `publications.json` with data exported from:

- Google Scholar (BibTeX → convert to JSON)
- OR use tools like:
  - Publish or Perish
  - scholar.py

## Optional Automation

Use GitHub Actions to auto-update publications.
